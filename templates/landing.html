SUST Project Archive

Arcxival is a project and resource managemnet archive.

Arcxival contains seperate layouts for Teachers and Students to maintain, manage and store projects, files, resources of any kind.
Visit the archive to check out the projects and get an idea about all the research works happening in SUST.

Intuitive Student Dashboard

Arcxival connects student projects with teachers for easy control.
See all your projects in one place divided by courses and fields. Search and update files anytime anywhere.




One stop place for all your projects. Contains a curated list of projects by Course, Session or Group.

Flexibility for Teachers

The Teacher Dashboard provides access to all Courses and Sessions maintained by the Teacher. Arcxival provides seperate panels to Create , Delete Sessions and Courses. This provides ease of access and managemnet to Teachers.




Contains seperate panels to Create new Courses, Sessions
and end running ones.

Upload Everything

Arcxival has a Drag and Drop file uploading mechanism with live thumbnails, size, data chunking and cancellation options.
